Explore Commands

Discover what Rimura can do with slash commands.

/ban [user][reason]

Bans a member permanently.

/mute [user] [duration]

Timeouts a user for specified time.

/clear[amount]

Purges up to 100 messages instantly.

/lockdown [channel]

Prevents non-admins from sending messages.

/play [song]

Plays a song or adds to queue.

/skip

Skips the current playing track.

/filters [type]

Applies bassboost, nightcore, etc.

/lyrics

Fetches lyrics for current song.

/gstart [time] [prize]

Starts a new giveaway.

/greroll[id]

Picks a new winner randomly.

/gend [id]

Force ends an ongoing giveaway.

/glist

Shows all active giveaways.

Got Questions?

Everything you need to know about Rimura.

How do I invite Rimura to my server?
Click any of the "Add to Discord" buttons on this website. You need to have "Manage Server" permissions in the server you wish to add Rimura to.
Does Rimura cost money?
Rimura is 100% free to use for all core features including Moderation, Giveaways, and standard Music. We offer an optional premium tier for 24/7 audio filters.
